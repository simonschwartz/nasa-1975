# Agency Name

The full agency name (National Aeronautics and Space Administration) is used, usually in conjunction
with the worm logo, to identify the agency as a total entity.

<div style={{ margin: '3rem 0' }}>
    <AgencyName size="xl" style={{ marginBottom: '2rem' }} />
    <AgencyName size="lg" style={{ marginBottom: '2rem' }} />
    <AgencyName size="md" style={{ marginBottom: '2rem' }} />
</div>

### Stacked

The agency name can be displayed stacked.

<StoryContainer>
    <AgencyName size="md" stacked />
</StoryContainer>

### Inline

The agency name is displayed inline by default.

<StoryContainer>
    <AgencyName size="md" />
</StoryContainer>

### Center Identification

To identify any of the NASA centers the agency name is shown in conjunction with the full center
name (John F. Kennedy Space Center) as shown below.

<StoryContainer>
    <AgencyName size="md" centerName="John F. Kennedy Space Center" />
</StoryContainer>

### On light background

Against a light background, the agency name should be shown in black.

<StoryContainer>
    <AgencyName size="md" centerName="John F. Kennedy Space Center" />
</StoryContainer>

### On dark background

Against a very dark color background, the agency name should always be shown in white.

<StoryContainer background="dark">
    <AgencyName size="md" centerName="John F. Kennedy Space Center" theme="light" />
</StoryContainer>

### API

```javascript
import { AgencyName } from 'nasa-1975';
```

<Props of={AgencyName} />

Any other properties supplied will be spread to the root element.
